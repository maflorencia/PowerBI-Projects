# PowerBI-Projects
#### 1- Proyecto Final Airbnb es el trabajo final realizado en equipo para el curso Data Analytics dicatado por Coderhouse. El objetivo fue analizar el mercado en ciudades de Brasil y México y realziar hallazgos que permitan mejorar oportunidades de negocio. Las etapas abarcaron desde la manipulación y la gestión de la base de datos en SQL hasta la posterior transformación de tablas, cálculos y creación de tablero en PowerBI.
